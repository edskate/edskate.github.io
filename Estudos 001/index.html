O que é um servidor web?

Todo site deve ser Hospedado em um servidor, muitas vezes chamado de host.

A sua função é enviar as páginas para os "clientes" que as solicitam.

O servidor web é um software especifico que você deve ter instalado no servidor.

O servidor é umcomputador normal, habitualmente com grande capacidade de armazenamento, que fica permanentemente ligado á internet por meio de uma linha telefõnica dedicada a uma rede.

O servidor web faz com que esse computador "entenda" as solicitações de páginas recebidas da internet e as envie para os usuários do mundo todo.

Como o objetivo desse curso é lhe passar os básicos do HTML, não será necessario hospedar os sites que você criar em um servidor web.

Porém, para trabalhar com linguagem de programação.
