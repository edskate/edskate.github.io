CSS border: Applying border styles to an element!


In CSS, the CSS border is part of the property set of HTML elements and is used to add a border around your content.
The border is located between the padding and the margin, and is part of the structure that makes up the “box model”,
which means a CSS box model formed by the content of the element and the padding, border and margin properties.

What are the border property values?

Setting the border width: border-width

The border-width property defines how big the border will be.
It can be defined using different units of measurement. See the possible syntaxes below:

Lenght. 👇

Indicates a size for the border, which can be absolute or relative values, including:

👆
  • px: numerical value in pixels
  • em: corresponds to twice the size of the font used in the element;
  • rem: indicates twice the font size of the root element;
  • vh and vw: correspond to the height (height) and width (width) of the screen area (viewport).

    • Example:

    Thin

    Setting the border-width property equal to “thin” makes the border value equal to 1px for all sides.
