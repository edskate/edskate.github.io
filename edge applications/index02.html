Hidden

The hidden value means that there is a border around the element, however, it will not be visible on the page. This is a useful feature for adding more space to an element without making the border visible.

border-style: hidden;

Dashed.

The dashed style represents a dotted line around the element.

Dashed.

Dotted.

The dotted style is represented by a line of circular dots around the element.

Dotted

Double.

In the double style, two lines are drawn around the element. It is important to say that the browser does the calculation to determine the thickness of each line. However, the sum of the two lines and the space between them correspond to the value defined in the border-width property.

Double.

Groove.

The groove style adds a color effect to the edge, which makes the element look like a pressed button.

Groove.
